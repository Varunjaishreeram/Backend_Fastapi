Welcome to the FastAPI Application

You can test this API using Postman with the following endpoints:

1. Create a Student

POST: /students

Request Body:

{
    "name": "Student 1",
    "age": 25,
    "address": {
        "city": "New York",
        "country": "USA"
    }
}

Example Response:

{
    "id": "603f1e4cfa0e9d44d87d3e2f"
}

2. Get All Students

GET: /students

Filters:

  • Filter by Country: /students?country=USA
  • Filter by Age: /students?age=25
  • Filter by Country and Age: /students?country=USA&age=25

Example Response:

{
    "data": [
        {
            "name": "Student 1",
            "age": 25,
            "_id": "603f1e4cfa0e9d44d87d3e2f",
            "address": {
                "city": "New York",
                "country": "USA"
            }
        }
    ]
}

3. Update a Student

PATCH: /students/{id}

Request Body:

{
    "age": 26
}

Example Response:

{
    "detail": "Student updated successfully"
}

4. Delete a Student

DELETE: /students/{id}

Example Response:

{
    "detail": "Student deleted successfully"
}
